# @khanenyao/rszr

[RSZR](https://github.com/khanenyao/rszr) - is a zero dependencies resize component. User can interact with with handles (controls) for increasing or decreasing the width or height of a container. As child can be any component.

Implemented with [React](https://reactjs.org) & [TypeScript](https://www.typescriptlang.org).

## Resizer props

| name     | type                 | required | default                                                                                |
| -------- | -------------------- | -------- | -------------------------------------------------------------------------------------- |
| children | ReactNode            | -        | -                                                                                      |
| controls | ResizerControlType[] | -        | [ResizerControlType.RIGHT, ResizerControlType.RIGHT_BOTTOM, ResizerControlType.BOTTOM] |

### Sidebar

<Story id="example-resizer--sidebar" />

```jsx
import React from "react";
import { Resizer, ResizerControlType } from "@khanenyao/rszr";
import "./sidebarExampleStyles.css";

const Example = () => {
  return (
    <div className="sidebar-example">
      <Resizer controls={[ResizerControlType.RIGHT]}>
        <nav>
          <ul>
            <li>Nav link one</li>
            <li>Nav link two</li>
            <li>Nav link three</li>
          </ul>
        </nav>
      </Resizer>
      <div className="container">content</div>
    </div>
  );
};

export default Example;
```
